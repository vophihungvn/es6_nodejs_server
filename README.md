Nodejs server write in ES6

Your first run
----------------------------------------------------------------------------
Install package:
```
npm i
```
For your development mode:
```
npm i --dev
```

Migration database: 
```
sequelize db:migrate
```
Find more information [here](/docs/database.md)

For testing:
Install `mocha` package globaly
```
npm i -g mocha
```

and run test
```
npm test
```
